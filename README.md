# RFIDjukebox

Personal project for my very own home cooked RFID jukebox via esp32. 

This is my first "proper" project outside of something in CS50x or something school related.
It includes streaming links associated to the different tag IDs that, once the esp32
connects to wifi, it will begin streaming while the tag continues to be scanned
by the reader. It stops playback when the tag is removed.

The project in it's entirety (3D modeling, painting, testing etc.) took a few months,
but the software development took around 15-20 hours alongside my schoolwork.
